# About

JavaScript/ES8 menu library. Simplest as possible.

# Features

- Vanilla JavaScript/ES8
- No dependencies

# Build minified bundle file

```bash
npm install
npm run build
```
Note: This is browser-centric vanilla JavaScript library, npm is only used to minify and bundle files.
